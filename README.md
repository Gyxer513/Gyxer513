## 🚀 What I Build

**Hotel Smart-TV SaaS Platform (acquired)** — multi-tenant IPTV/hospitality platform: guest-facing TV interface, hotel admin panel, content & billing management. Built solo from zero to a commercial product, registered with Rospatent, and **sold under a multi-year contract**. Stack: Kotlin (Android TV) · React (TV UI) · Vue 3 (admin) · NestJS · PostgreSQL.

**On-Prem Medical AI Assistant (B2B, ~300 physicians)** — RAG-based knowledge assistant for a hospital: retrieval over indexed medical course materials, delivered as a Chrome MV3 extension rolled out via Active Directory GPO. Fully air-gapped inference: **Qwen2.5-72B (4-bit) on hospital hardware**. Stack: FastAPI · Qdrant · vLLM · Chrome Extensions API.

**"Kombain" — Self-Hosted Multi-Agent AI System** — orchestrated agent platform (SecOps, coding, and general-purpose agents) with a hybrid LLM provider layer, RAG memory, and Telegram interface. Stack: Pydantic AI · LiteLLM · LlamaIndex + Qdrant · aiogram.

**Fintracker — Personal Finance "CFO"** — live product with a Telegram Mini App frontend: loan overpayment analysis, early-repayment scenario comparison, deposit projections with capitalization. Precise money math on integer minor units (no floats), annuity & differentiated amortization engines. Stack: FastAPI · Vue 3 · Telegram Mini App.

**Vaccination Planning System (MedTech)** — immunization scheduling engine for clinics: patient vaccination plans computed on the fly from age, history, and the active immunization calendar. National calendar with regional extensions modeled as an inheritable catalog chain, catch-up vaccination logic, pediatric/adult scoping, sex-specific schedules, and matching of legacy medical-record codes against modern schedule definitions. Stack: TypeScript · tRPC · Prisma · PostgreSQL · Zod · Keycloak SSO · Turborepo monorepo.

**Corporate Internal Portal** — monorepo with employee self-service portal and admin panel: Active Directory SSO, payslip access, document storage, procurement registry. Stack: React 18 + TypeScript · NestJS · MongoDB · Docker Compose · GitLab CI.

**Internal Web Services** — payslip automation with ERP sync, procurement & contract registries. Full CI/CD via GitLab: multi-stage Docker builds, one-click deploy & rollback pipelines.
