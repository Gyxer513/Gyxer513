## 🚀 What I Build

**ERP/CRM Platform (MedTech)** — end-to-end automation of business processes in healthcare: finance, medical data, supply chain, payments, integrations with external services. Stack: Vue 3 + TypeScript + Vite + Django REST Framework + PostgreSQL.

**Internal Web Services** — payslip automation with ERP sync, procurement & contract registries. Full CI/CD via GitLab: multi-stage Docker builds, one-click deploy & rollback pipelines.

**B2B SaaS (MedTech)** — healthcare service cost calculation platform.

**Corporate Internal Portal** — monorepo with employee self-service portal and admin panel. Features: Active Directory SSO, payslip access, document storage, procurement registry, announcements. Stack: React 18 + TypeScript + Vite · NestJS · MongoDB · Docker Compose.
CI/CD: GitLab CI with SSH deploy pipeline.
